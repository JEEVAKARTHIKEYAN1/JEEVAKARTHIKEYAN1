### Tablet Layout (768px-1199px)

- Sidebar moves above main content
- KPI strip remains 4-column

### Mobile Layout (<768px)
- KPI strip becomes 2x2 grid
- Overview sections stack vertically
- Navigation tabs may require horizontal scrolling
